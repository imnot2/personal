登录接口
HTTP请求方式
GET/POST
请求URL
http://221.122.112.136:8080/sms/mt.jsp?cpName=用户账号&cpPwd=用户密码&phones=号码&msg=内容
请求参数
参数名称 类型 是否必须 描述
cpName 用户账号 是 用户账号
cpPwd 用户密码 是 用户密码
phones 接收号码 是 同时发送给多个号码时,号码之间用英文半角逗号分隔
GET方式每次最多可以提交50条号码
POST方式每次最多可以提交10000个号码
msg 短信内容 是 发送内容需要用gbk编码进行UrlEncode,如果采用post方式，需要用GBK编码做两次URLEncode
发送内容后面添加签名，例如：【交通银行】,否则会失败
spCode 用户流水号 否 用户自定义流水号(20位以内数字)
返回结果
参数名称 类型 描述
返回描述 String 发送成功返回0，如果发送不成功，则返回“ERROR&&对应的错误信息”
